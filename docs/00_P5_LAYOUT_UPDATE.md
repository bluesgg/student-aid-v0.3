# P5页面布局修正说明

## 布局变更

**从**：左右两栏（PDF阅读器 | 贴纸栏+问答区上下分布）

**改为**：左中右三栏（PDF阅读器 | 贴纸栏 | 问答区）

## 新布局结构

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 面包屑 + 文件名                              [配额提示: 87/150]          │
├──────────────────────┬──────────────────────┬────────────────────────────┤
│                      │  [✓ Explain page]    │  [✓ Ask question]          │
│                      ├──────────────────────┼────────────────────────────┤
│   左侧：PDF阅读器     │   中栏：贴纸栏        │   右栏：问答区              │
│   - 分页/滚动模式     │   - 自动讲解贴纸      │   - 历史问答                │
│   - 缩放/页码跳转     │     (浅色背景)       │   - 提问输入框              │
│   - 记忆lastReadPage  │   - 手动讲解贴纸      │   - 总结卡片                │
│                      │     (深色背景)       │   - 文档/章节总结           │
│                      │   - 支持折叠/展开     │                            │
│                      │   - 追问链(最多3层)  │                            │
└──────────────────────┴──────────────────────┴────────────────────────────┘
```

## 布局比例建议

- 左侧PDF阅读器：40-50%宽度
- 中栏贴纸栏：25-30%宽度
- 右栏问答区：25-30%宽度

## 优势

1. **空间利用更高效**：贴纸和问答可同时可见，无需上下滚动切换
2. **视觉对比更清晰**：两个AI功能区物理分离
3. **工作流更顺畅**：阅读PDF→查看贴纸→提问，视线自然从左到右流动
4. **响应式考虑**：小屏幕可改为标签页切换（贴纸tab | 问答tab）
